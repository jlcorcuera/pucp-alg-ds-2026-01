## 📖 Course Description

This repository contains **in-class exercises** developed during the lectures of the **Algorithms and Data Structures** course offered at PUCP during the 2026-1 academic semester.

The course is of a theoretical-practical nature, whose purpose is for the student to understand the implementation and application of fundamental algorithms and data structures. Topics developed include asymptotic analysis, brute force, recursion, lists, stacks, queues, divide and conquer, binary trees, binary search trees, and graphs.

## ⚙️ Getting Started

### Prerequisites

Ensure you have the following tools installed:

- **Git** ≥ 2.30
- **C++ Compiler** (e.g., GCC/g++ ≥ 11, Clang)
- **CLion** (the official IDE we will use for this course)

### Clone the Repository

```bash
git clone https://github.com/jlcorcuera/pucp-alg-ds-2026-01.git
cd pucp-alg-ds-2026-01
```

### Running Code Examples

Each lecture/lab folder will contain specific setup and execution instructions if needed. In general, to compile and run the exercises using g++:

```bash
g++ -O2 -o solution solution.cpp
./solution
```
